# 문자열 자료형

- 문자열(String) : 문자, 단어 등으로 구성된 집합
- 따옴표(",')로 둘러싸여 있으면 모두 문자열

```python
"Life is too short, You need Python"
"a"
"123"
```

## 문자열을 만들고 사용하기

1. 큰 따옴표로 양쪽 둘러싸기

```python
"Hello World"
```

2. 작은 따옴표로 둘러싸기

```python
"Pyhon is fun"
```


3. 큰 따옴표 3개를 연속으로 써서 양쪽 둘러싸기

```python
"""Life is too short, You need pyhon"""
```

4. 작은 따옴표 3개를 연속으로 싸서 양쪽 둘러싸기

```python
'''Life is too short, You need python'''
```
